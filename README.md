# Business Sales Analysis Capstone Project

## Overview
This project performs an end-to-end data analysis of business sales data to identify relationships between marketing spending and revenue.

## Objectives
- Analyze sales performance
- Identify impact of marketing spend
- Generate business insights
- Provide data-driven recommendations

## Dataset
The dataset contains 500+ records including:

- Date
- Marketing Spend
- Sales
- Region
- Customer Count

Install required libraries:

pip install -r requirements.txt

## Key Analysis Techniques

- Exploratory Data Analysis
- Correlation Analysis
- Hypothesis Testing
- Linear Regression

## Key Findings

- Strong correlation between marketing spend and sales
- Marketing significantly impacts revenue
- Marketing explains most of sales variability

## Business Recommendations

- Increase marketing investment strategically
- Optimize marketing channels
- Use predictive analytics for sales forecasting
